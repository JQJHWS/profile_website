# 分享给非程序员用户指南

## 方式一：在线部署（最推荐）

将网站部署到免费平台，用户只需访问网址即可使用。

### 选项 A：Vercel 部署（最简单）

1. 访问 https://vercel.com 并注册账号（可用 GitHub 登录）
2. 点击 "Add New..." → "Project"
3. 选择 "Import Git Repository" 或直接拖拽整个项目文件夹
4. 点击 "Deploy" 等待部署完成
5. 获得免费网址如：`https://resume-builder-xxx.vercel.app`
6. 将网址分享给朋友即可！

### 选项 B：Netlify 部署

1. 访问 https://netlify.com 并注册账号
2. 点击 "Add new site" → "Deploy manually"
3. 将 `dist` 文件夹拖拽到页面上
4. 等待部署完成，获得免费网址
5. 分享网址给朋友

### 选项 C：GitHub Pages 部署

1. 在 GitHub 创建仓库
2. 上传项目代码
3. 在仓库 Settings → Pages 选择分支
4. 等待自动部署完成

---

## 方式二：打包发送（需对方安装 Node.js）

### 步骤 1：打包项目
将整个 `profile_creater` 文件夹压缩成 ZIP 文件发送给对方。

### 步骤 2：对方需要做的
1. 安装 Node.js：访问 https://nodejs.org/ 下载安装
2. 解压收到的 ZIP 文件
3. 双击 `start.bat` 启动
4. 浏览器访问 http://localhost:5173

---

## 方式三：打包成独立应用（无需安装任何东西）

使用 Electron 打包成 .exe 文件，对方双击即可运行。

需要额外安装依赖并打包，适合需要分发给多人的场景。

---

## 推荐方案

| 场景 | 推荐方式 |
|------|----------|
| 分享给 1-2 个朋友 | Vercel/Netlify 在线部署 |
| 分享给多人使用 | 在线部署 + 分享网址 |
| 朋友有一定电脑基础 | 打包 ZIP + 安装 Node.js |
| 完全不懂电脑的用户 | 在线部署（最简单） |

---

## 快速部署命令

如果您已安装 Vercel CLI：
```bash
npm i -g vercel
vercel
```

如果您已安装 Netlify CLI：
```bash
npm i -g netlify-cli
netlify deploy --prod --dir=dist
```
